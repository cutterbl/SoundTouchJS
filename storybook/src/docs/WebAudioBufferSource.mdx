# WebAudioBufferSource

**What is this?**

This is a simple adapter that lets you use a browser `AudioBuffer` (a chunk of sound) with SoundTouchJS’s processing tools.

**When would I use it?**

If you have audio loaded in the browser and want to process it (change pitch, speed, etc.), use this class to connect it to the rest of the toolkit.

`WebAudioBufferSource` adapts a browser `AudioBuffer` to the extraction contract expected by `SimpleFilter`.

## Import

```ts
import { WebAudioBufferSource } from '@soundtouchjs/core';
```

## Constructor

```ts
new WebAudioBufferSource(buffer: AudioBuffer)
```

## Public API

- dualChannel (getter)
- position (getter/setter)
- extract(target, numFrames?, position?)

## Notes

- Output is interleaved stereo.
- Mono sources are duplicated to both channels.
