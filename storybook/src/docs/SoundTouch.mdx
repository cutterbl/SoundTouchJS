# SoundTouch

**What is this?**

This is the main “engine” that lets you change pitch, tempo, and speed of audio in one place. It combines all the building blocks (like stretching and transposing) so you don’t have to wire them up yourself.

**When would I use it?**

If you want to change how fast something plays, shift the pitch, or adjust the key—all in one step—this is the class to use.

`SoundTouch` is the main high-level engine that combines `Stretch` and `RateTransposer` for pitch, tempo, and rate manipulation.

## Import

```ts
import { SoundTouch } from '@soundtouchjs/core';
```

## Constructor

```ts
new SoundTouch({
  sampleRate?: number,
  sampleBufferType?: 'fifo' | 'circular',
  sampleBufferFactory?: SampleBufferFactory,
  interpolationStrategy?: RateTransposerInterpolationStrategy,
})
```
