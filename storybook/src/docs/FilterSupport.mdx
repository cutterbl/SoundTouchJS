# FilterSupport

**What is this?**

This is a helper class that makes it easier to move sound through your processing pipeline. Think of it as a “traffic manager” that keeps the audio flowing smoothly from one stage to the next.

**When would I use it?**

If you’re building custom audio effects or want to control how sound is buffered and processed, this class gives you more control.

`FilterSupport` wraps a SamplePipe and provides utility flow for filling and processing output buffers.

## Import

```ts
import { FilterSupport } from '@soundtouchjs/core';
```

## Constructor

```ts
new FilterSupport(pipe: SamplePipe)
```

## Public API

- pipe (getter)
- inputBuffer (getter)
- outputBuffer (getter)
- fillInputBuffer(numFrames)
- fillOutputBuffer(numFrames?)
- clear()

## Notes

- fillInputBuffer is intended to be overridden.
- fillOutputBuffer repeatedly fills/processes until enough output is available or input runs out.
