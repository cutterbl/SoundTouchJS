# Lanczos8 Strategy (High Quality)

**What is Lanczos8?**

Lanczos8 is a fancy math method for making audio sound smooth and natural when you change its speed or pitch. It’s the default in SoundTouchJS because it usually sounds the best—like a high-res photo instead of a pixelated one.

**When should I use it?**

- If you want your audio to sound as close to the original as possible (music, podcasts, anything with voices or instruments)
- If you don’t mind using a bit more CPU for better sound

**What’s the catch?**

- It uses more processing power than simpler methods, so it’s not always the best for live effects or on very slow devices.

**What does the `radius` parameter do?**

- Think of `radius` as how “wide” the smoothing is. Bigger numbers = smoother sound, but also more CPU. Try 4 to start (range: 2–8).

**How do I use it?**

```ts
import { SoundTouch } from '@soundtouchjs/core';

const soundTouch = new SoundTouch({
  interpolationStrategy: {
    id: 'lanczos8',
    params: { radius: 4 }, // 4 is a good default
  },
});
```

**Tip:** If you’re not sure, just use the defaults. You can always experiment with the `radius` value and listen for what sounds best in your app.
